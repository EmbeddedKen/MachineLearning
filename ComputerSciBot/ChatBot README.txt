ChatBot.py - a Computer Science Q&A ChatBot
ChatBot.py is knowledgable in basic CompSci Questions.
It's dependencies include: 'chatterbot', 'python3.6.x'

*NEW: Now has basic WinForms GUI. In order to run, copy the 'ChatBot_Exe' folder to 'C:\'.
This is the only place it can run currently due to relative file path issues.

----------------------------------------------------
You can ask him questions like...

'What is an Object File?' and
'Why do I have a Stack Overflow?'

ChatBot.py uses a short list of hand-written answers to common questions, and the chatterbot API uses
Bayesian machine learning to determine which of the following list answers is believed to be the most
accurate match for the given input string.
